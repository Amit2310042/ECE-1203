## **signed **
Declares that a variable can hold both positive and negative values. Typically used with char or int. int is signed by default, but it’s useful for clarity.

## **Example :**
```C
signed int temp = -15;
char letter = 'A';
signed char signChar = -20;  // explicitly signed

```
